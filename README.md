# 💰 Personal Expense Tracker (CLI)

A lightweight, command-line personal expense tracker built entirely in Python. This tool allows users to quickly log daily expenses, categorize their spending, and generate automated summaries to track financial habits.

Built with a focus on modular code, file handling, and foundational data analysis.

## ✨ Features

* **Add Expenses:** Quickly log the amount, category, and date of an expense.
* **Data Persistence:** Automatically saves all records to a local `expenses.csv` file.
* **Automated Summaries:** Calculates total monthly spend, identifies the highest spending category, and computes the average daily spend.
* **Export Reports:** Generates clean `.txt` reports of your expense history.

## 🛠️ Tech Stack & Skills Demonstrated

* **Language:** Python 3.x
* **Libraries:** `csv`, `os`, `datetime`, `collections`
* **Concepts:** File handling (CSV writing/reading), basic data analysis & aggregation, modular function design, error handling.
